CTR.Tool - вставляется в конце стандартных функций крафта (Recipes.addShaped и Recipes.addShapeless). Функция ищет в рецепте айди инструмента, добавляемого специальной функцией, и тратит его прочность. Инструментов в рецепте может быть сколько угодно. В следующих функциях крафта, добавляемых библиотекой, ее использовать необязательно.

CTR.addTool(i, d) - регистрирует инструмент в библиотеке. i - айди, d - прочность.

CTR.RegisterHammer(name, data) - регистрирует молот. name - материал, data - прочность. Рецепт для данного инструмента регистрируется автоматически (если указанный материал существует)

CTR.RegisterTungstensteel(name, data) - регистрирует напильник. name - материал, data - прочность. Рецепт для данного инструмента регистрируется автоматически (если указанный материал существует)

CTR.RegisterMortar(name, data) - регистрирует ступку. name - материал, data - прочность. Рецепт для данного инструмента регистрируется автоматически (если указанный материал существует)

CTR.RegisterCutter(name, data) - регистрирует ножницы. name - материал, data - прочность. Рецепт для данного инструмента регистрируется автоматически (если указанный материал существует)

CreateRecipeWithTool(result, ing, tr, tool_arr) - регистрирует форменный рецепт с зарегистироваными инструментами. Принимает такие же аргументы, как и обычный addShaped. tool_arr представляет из себя массив с массивами используемых инструментов (не более двух видов)

ReplaceRecipeWithTool(result, ing, tr, tool_arr) - пересоздает форменный рецепт с зарегистироваными инструментами. Принимает такие же аргументы, как и обычный addShaped. tool_arr представляет из себя массив с массивами используемых инструментов (не более двух видов)

Для каждого вида инструмента используется своя буква, регистрировать которую в функции не нужно: 'h' - молот, 't' - напильник, 'm' - ступка, 'n' - ножницы.
CreateShapelessRecipeWithTool(result, ing, tool) - регистрирует бесформенный рецепт с зарегистироваными инструментами. Принимает такие же аргументы, как и обычный addShapeless. tool_arr представляет из себя массив используемого инструмента. 


ReplaceShapelessRecipeWithTool(result, ing, tool) - пересоздает бесформенный рецепт с зарегистироваными инструментами. Принимает такие же аргументы, как и обычный addShapeless. tool_arr представляет из себя массив используемого инструмента.

MAPI.Register(name, struc) - регистрирует многоблочную структуру за выбранным тайлом. name - объект тайла, struc - массив структуры.

MAPI.Rotate(struc) - создает четыре варианта положения в пространстве выбранного массива структуры

MAPI.Corpus(this) - необходимо поместить в тик машины. Для проверки целостности структуры используйте значение this.data.MAPIact (возвращает true или false)

ATMech.FurnaceRecipe(obj) - создает рецепт для многоблочных печей

ATMech.Fuel(obj) - создает топливо для многоблочных печей

ATMat.MaterialRegister(name, obj, obj1) - регистрирует материал. Пример obj: {isDust:true, isIngot:true, isPlate:true, isNugget:true, isSmallDust:true, isLittleOre, isBolt:true, isModule:true, generateRecipes:true}. obj1 необходим для создания рецептов в печах. Пример obj1: {temp:1600, long:240}.
